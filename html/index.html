2. 세팅 방법

  1. 1. C드라이브에 소나큐브 압축을 푼 뒤 'bin' 폴더 > 'windows-x86-64' 폴더로 들어가줍니다.

  2. 2. 'windows-x86-64' 폴더 안에서 터미널을 실행 시킨 후 StartSonar.bat을 입력 해줍니다.

  3. 3. 위 이미지와 같이 프로세스가 뜨면 웹 브라우저에서 localhost:9000을 입력 해 접속합니다.

  4. 4. 접속하면 로그인창이 나오는데, 초기 계정 정보는 admin / admin 이며 비밀번호는 변경 할 수 있습니다.

  5. 5. 로그인 하면 위 이미지와 같은 화면이 나오는데, 여기서 Manually를 눌러줍니다.

  6. 6. 프로젝트 이름과 키 값을 적어줍니다. 프로젝트 키 값은 프로퍼티에 적어줘야 하기 때문에 잊어버리시면 안돼요!!!!!

  7. 7. 소나스캐너를 작업하시는 폴더에다 압축을 풀어주시고, 같은 폴더에 sonar-project.properties 파일을 생성해줍니다.

  8. 8. sonar-project.properties 파일안에 값을 넣어줍니다.

    sonar.projectKey는 6번에서 만든 키 값을 넣어주고,
    sonar.password는 변경하신 비밀번호를 넣어주시면 됩니다.
    sonar.exclusions는 특정 경로나 파일을 예외 처리 할 수 있습니다.
    여러 개 예외 처리하려면 콤마(,)로 구분하고 앞뒤로 쌍따옴표(")를 붙여줍니다.

    > 사용 가이드

  9. 9. 작업하시는 폴더에서 마우스 우클릭 후 git Bash Here를 눌러줍니다.

  10. 10. 터미널이 열리면 ./sonar-scanner/bin/sonar-scanner.bat 를 입력합니다.

  11. 11. 입력한 후 접속했던 localhost:9000을 새로고침 하시면 전체 프로젝트의 상태가 나옵니다.

  12. 12. 프로젝트의 상세 내용이며 숫자를 클릭하시면 프로젝트의 이슈 상태를 볼 수 있습니다.

  13. 13. 프로젝트의 이슈 상태이며 여기 나온 가이드를 참고해서 수정하시면 됩니다.
